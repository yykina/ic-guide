# 清华大学 王诚/刘卫东：计算机组成原理（国家精品）

## 课程简介

- 所属大学：清华大学
- 先修要求：数字逻辑
- 编程语言：汇编、Verilog
- 授课教师：王诚、刘卫东
- 课程难度：🌟🌟🌟
- 预计学时：约 50 讲 / 40 学时

清华王诚/刘卫东主讲的国家精品课，配套 **TEC-2000 教学机大作业**——“奋战三星期做台计算机”，是中文世界最经典的组成原理课。覆盖数据表示、运算器、控制器、存储系统、I/O 全套硬件原理。

适合在 CSAPP 之前作为硬件视角的中文打底。
