# 中国科学院大学 胡伟武：计算机体系结构（龙芯）

## 课程简介

- 所属大学：中国科学院大学
- 先修要求：计算机组成原理
- 编程语言：LoongArch 汇编、Verilog
- 授课教师：胡伟武（龙芯总设计师）
- 课程难度：🌟🌟🌟🌟
- 预计学时：约 60 课时

龙芯总设计师胡伟武主讲，结合自主 CPU 研发实战讲体系结构。配套开源教材《计算机体系结构基础》（第三版），全书 GitBook 免费可读。

是中文圈“工业一线视角”讲体系结构的代表课，与学院派课程互补。
