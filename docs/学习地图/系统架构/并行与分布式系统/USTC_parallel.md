# 中国科学技术大学：并行计算（国家精品）

## 课程简介

- 所属大学：中国科学技术大学
- 先修要求：操作系统、计算机体系结构
- 编程语言：C / MPI / OpenMP
- 授课教师：中科大并行计算团队
- 课程难度：🌟🌟🌟
- 预计学时：约 40 小时

国家级精品课，把**并行机体系结构 / 并行数值算法 / 并行编程**（MPI/OpenMP）三者有机整合，是国内最经典的并行计算入门课。
