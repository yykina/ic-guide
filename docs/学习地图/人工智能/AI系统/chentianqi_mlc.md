# CMU 陈天奇：机器学习编译（MLC 中文版）

## 课程简介

- 所属大学：CMU / Apache TVM 社区
- 先修要求：深度学习、Python、C++
- 编程语言：Python（TVM Unity）
- 授课教师：陈天奇
- 课程难度：🌟🌟🌟🌟
- 预计学时：11 讲（约 30 学时）

陈天奇 2022 夏开设的中文系统课，围绕**计算图 / 张量程序 / 算子库 / 硬件指令**四大抽象讲透 TVM Unity 与 MLC 思想。配套 Jupyter Notebook 实操，端侧 AI 部署中文教程。

与“智能计算系统”主题互补——AICS 讲完整软硬栈，MLC 聚焦编译器抽象。
