# 张志华：统计机器学习

## 课程简介

- 所属大学：上海交通大学 / 北京大学
- 先修要求：高等概率论、线性代数、机器学习导论
- 编程语言：无（理论课）
- 授课教师：张志华
- 课程难度：🌟🌟🌟🌟🌟
- 预计学时：约 30+ 讲

张志华教授经典理论课。从**概率不等式 / 指数族 / 共轭先验 / Wishart 分布**到 **RKHS / 谱方法**的数学化机器学习。是中文圈最接近 CS229M / 10-708 风格的理论课。

适合数学/统计/CS 研究生作为机器学习数学基础的深入课。
