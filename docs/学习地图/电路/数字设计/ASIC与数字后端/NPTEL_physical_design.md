# NPTEL：VLSI Physical Design（IIT KGP）

## 课程简介

- 所属大学：Indian Institute of Technology Kharagpur (IIT KGP)
- 授课教师：Prof. Indranil Sengupta
- 先修要求：数字逻辑基础、数据结构与算法（图论）
- 课程难度：🌟🌟🌟🌟
- 预计学时：64 讲（约 64 小时）

布局布线是 EDA 算法研究的核心战场。这门课系统讲授 **VLSI 物理设计全流程的算法本质**：划分（Kernighan-Lin / Fiduccia-Mattheyses）、布图规划（Slicing Floorplan / BSG）、布局（模拟退火 / 解析布局）、布线（迷宫算法 / A* / Steiner 树）、时序约束与静态时序分析。每个章节从问题定义出发，推导复杂度，讲解经典启发式，是学术界最系统的物理设计算法教程之一。

配合 B 站双语字幕版，中文理解障碍几乎为零。
