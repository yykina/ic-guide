# Nandland: Learn FPGA, Verilog & VHDL

## 课程简介

- 所属大学：无（独立教育平台）
- 先修要求：无（专为完全零基础设计）
- 编程语言：Verilog / VHDL
- 课程难度：🌟🌟
- 预计学时：30～60 小时

Nandland 是由 Russell Merrick 创建的 FPGA 入门教育平台，以“让 FPGA 学习对初学者友好”为宗旨，是英语互联网上评价最高的免费 FPGA 入门站点之一。

与其他教程相比，Nandland 最显著的特点是 Verilog 与 VHDL 同等覆盖（许多平台只讲其一），且讲解风格极为简洁直白，配有大量图示和动画说明，适合从未接触过硬件描述语言的同学从零建立概念。平台提供结构化的 FPGA-101 基础课程、编程教程、配套教学视频，以及一本 300 页的入门书籍《Getting Started with FPGAs》。如果希望有配套硬件实践，可选购 Nandland 自家的 Go Board 开发板，但所有教程内容均可通过软件仿真完成，不强制购买硬件。

推荐作为 HDLBits 刷题前的前置热身，或与 HDLBits 交替使用——Nandland 帮助理解“为什么这样写”，HDLBits 提供大量“动手练”的机会。
