# 浙江大学：玩转数字系统设计（国家精品）

## 课程简介

- 所属大学：浙江大学
- 先修要求：数字逻辑基础、Verilog 入门
- 编程语言：Verilog
- 授课教师：浙大数字系统团队
- 课程难度：🌟🌟🌟
- 预计学时：一学期 MOOC（理论 + 实验融合）

浙大将“数字系统设计”+“实验”两课融合的国家级精品 MOOC，对标国际顶尖工科课程内容。理论与 Verilog 实践不脱节，从组合逻辑、时序逻辑一路推到处理器核设计。

比 HDLBits 更系统化的中文版替代，适合系统性学习数字系统设计而非仅练习 Verilog 语法。
