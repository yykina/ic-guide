# 高亚军：跟 Xilinx SAE 学 HLS

## 课程简介

- 授课教师：高亚军（Xilinx 亚太区高级应用工程师）
- 先修要求：Verilog 基础、C/C++ 基础
- 课程难度：🌟🌟🌟
- 预计学时：25 讲（约 15 小时）

国内 HLS 中文入门的事实标准。高亚军以 Xilinx SAE 身份讲授 Vivado HLS，深度远超官方 UG 文档：pragma 的效果与限制、接口综合（AXI4 / FIFO / BRAM 接口的 HLS 描述方法）、流水线与循环展开的相互作用、数据流优化（Dataflow pragma）。课程对每个 pragma 都有对比实验，直接演示改前改后的延迟和资源报告，是最快建立 HLS 工程直觉的中文资料。

注意：基于 Vivado HLS（旧工具链），原理与 Vitis HLS 通用，界面有差异。
