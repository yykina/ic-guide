# NPTEL CMOS Digital VLSI Design（IIT Roorkee）

## 课程简介

- 所属大学：印度理工学院鲁尔基分校（IIT Roorkee）
- 先修要求：数字逻辑基础、半导体器件物理基础（MOSFET 工作原理）
- 编程语言：无（仿真实验可选 SPICE / Cadence Virtuoso）
- 课程难度：🌟🌟🌟🌟🌟
- 预计学时：100 小时

NPTEL（印度国家技术增强学习计划）是全球最早免费开放工程课程视频的平台之一。本课程由 IIT Roorkee 微电子方向的 Sudeb Dasgupta 教授主讲，是 NPTEL 平台数字 VLSI 方向的代表性课程。理论推导严谨，内容覆盖完整的数字 CMOS 集成电路设计链路。课程全程英语授课，共 40 讲，适合有一定半导体器件基础的同学系统进阶。

课程核心内容涵盖：MOS 晶体管基础与 SPICE 模型 → CMOS 反相器与静态/动态逻辑门的电路分析 → 晶体管尺寸优化（sizing）与驱动能力设计 → 功耗分析（动态功耗、静态漏电）→ 时钟策略与锁存器/触发器设计 → 组合与时序逻辑电路设计 → SRAM 存储器单元设计。整体深度与复旦《数字集成电路设计原理》（MICR130029）高度吻合，是该课程最推荐的配套视频资源。

与 Weste & Harris《CMOS VLSI Design》教材（数字集成电路方向全球标准教材）结合使用效果最佳。两者在 CMOS 逻辑家族、时序分析与存储器设计章节几乎一一对应，课程结束后可直接过渡到芯片后端设计与综合工具链学习。
