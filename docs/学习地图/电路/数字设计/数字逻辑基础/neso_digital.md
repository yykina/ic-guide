# Neso Academy: Digital Electronics

## 课程简介

- 所属大学：独立教育频道（YouTube / B 站）
- 先修要求：无（高中数学）
- 编程语言：无
- 课程难度：🌟🌟🌟
- 预计学时：80 小时

Neso Academy 是 YouTube 上订阅量超过 300 万的工程教育频道，其《Digital Electronics》系列共 150+ 集，是全平台内容最系统、覆盖面最广的数字电路免费公开课。每集时长 10–20 分钟，讲解清晰，节奏友好，从布尔代数零基础出发，逐步覆盖所有数字电路核心主题。

课程完整涵盖：数制与编码（二进制、八进制、十六进制、BCD、格雷码）→布尔代数与逻辑化简（卡诺图 K-Map、Quine-McCluskey 法）→组合逻辑电路（半加器/全加器、多路复用器/解复用器、编码器/译码器）→时序逻辑电路（SR/JK/D/T 触发器、寄存器、计数器）→有限状态机（Mealy/Moore）→存储器（ROM/RAM 原理）→可编程逻辑器件（PLD/CPLD 基础）。内容与复旦《数字逻辑基础》（MICR130003）完全对应，且难度与进度更加平滑，是入门数字电路的最优视频资料。

B 站有完整搬运版（200 集）。
