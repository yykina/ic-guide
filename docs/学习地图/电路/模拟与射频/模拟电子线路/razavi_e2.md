# Razavi Electronics 2（UCLA）

## 课程简介

- 所属大学：UCLA（加州大学洛杉矶分校）
- 授课教师：Behzad Razavi
- 先修要求：[Razavi Electronics 1](razavi_e1.md)
- 课程难度：🌟🌟🌟🌟
- 预计学时：46 讲（约 46 小时）

衔接 Electronics 1 的进阶课程，覆盖：共源共栅（Cascode）结构、电流镜与偏置电路、差分对、频率响应（Miller 效应、主极点/次极点分析），以及负反馈理论与环路稳定性判据（Bode 图、相位裕度）。这些内容是模拟 IC 设计（CMOS 运放、LNA、VCO）的核心前置知识。
