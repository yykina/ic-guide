# 清华大学：模拟电子技术基础（华成英）

## 课程简介

- 所属大学：清华大学
- 先修要求：电路分析基础（KVL/KCL）、大学物理基础
- 编程语言：无
- 课程难度：🌟🌟🌟🌟
- 预计学时：120 小时

华成英教授是清华大学模拟电子技术课程的代表性教师，其主编的《模拟电子技术基础》（清华大学出版社）与童诗白版并列为国内使用最广泛的模拟电子教材。B 站上这套 189 讲的完整录像是国内最系统、最权威的中文模拟电子公开课之一，覆盖从半导体器件基础到运算放大器应用的完整知识体系。

课程内容涵盖：半导体器件（二极管、BJT、MOSFET）的物理机制与小信号模型→基本放大电路（共射/共基/共集）→多级放大与频率响应→集成运算放大器内部结构→负反馈理论→运放的线性与非线性应用→功率放大器→直流稳压电源。内容与复旦《模拟电子线路》（MICR130002）高度对应，是该课程的最优中文配套资料。

与 Razavi Electronics 1 & 2 相比，华成英课程采用 BJT 优先的讲授顺序（更接近国内考研方向），而 Razavi 以 MOSFET 为主线（更接近工业界实践）；两套课程互为补充，配合使用效果最佳。
