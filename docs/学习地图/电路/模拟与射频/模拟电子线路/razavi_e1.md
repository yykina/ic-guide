# Razavi Electronics 1（UCLA）

## 课程简介

- 所属大学：UCLA（加州大学洛杉矶分校）
- 授课教师：Behzad Razavi
- 先修要求：电路分析基础（KVL/KCL、戴维南/诺顿定理）、半导体物理基础
- 课程难度：🌟🌟🌟🌟
- 预计学时：45 讲（约 45 小时）

Razavi Electronics 1 从半导体载流子物理出发，系统讲授 PN 结与二极管电路、BJT 放大器的偏置与小信号模型、MOSFET 的工作原理与 I-V 特性，以及共源（CS）、共栅（CG）、共漏（CD）三种基本放大拓扑。Razavi 擅长以直觉建立电路物理图像再辅以严格数学分析，对模拟 IC 设计有极大帮助。

对应国内教材（谢嘉奎/童诗白体系）的前半段，可与复旦《模拟电子线路》（MICR130002）配合使用。学完后续接 [Electronics 2](razavi_e2.md)。
