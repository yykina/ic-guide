# 上海交通大学 郑益慧：模拟电子技术基础

## 课程简介

- 所属大学：上海交通大学
- 先修要求：电路分析基础（KVL/KCL、正弦稳态）
- 编程语言：无
- 授课教师：郑益慧
- 课程难度：🌟🌟🌟
- 预计学时：80 小时（全 52 讲）

B 站口碑顶级的中文模电课，4K 超清重制版。郑益慧老师把半导体器件的物理图像和小信号等效模型讲得非常透，被广泛认为讲课水平不输清华华成英版。课程覆盖半导体器件（二极管、BJT、MOSFET）及小信号模型、基本放大电路、多级放大与频率响应、负反馈、运算放大器应用、功率放大与直流稳压电源，与复旦《模拟电子线路》（MICR130002）知识点高度对应，是该课最顺手的中文配套视频。

与同目录清华华成英版二选一作主线即可。郑版器件物理图像更清楚，华成英版与童诗白教材绑定更紧，用哪本教材就配哪门课。
