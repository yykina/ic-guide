# 电子科技大学：模拟电子技术基础（何松柏等）

## 课程简介

- 所属大学：电子科技大学
- 先修要求：电路分析基础、大学物理基础
- 编程语言：无
- 课程难度：🌟🌟🌟🌟
- 预计学时：70 小时

电子科技大学《模拟电子技术基础》是国家级精品课程，由何松柏、吴涛、张彪、谢建志等多位教师联合主讲，共约 48 讲，B 站有完整公开录像。电子科大在模拟电路领域积淀深厚，该课程注重理论与工程实践并重，对模拟电路中的工程近似方法讲解尤为清晰，是国内同类课程中工程感最强的版本之一。

课程系统覆盖：半导体器件（二极管、BJT、JFET、MOSFET）→静态工作点分析与偏置→小信号等效电路模型→单级/多级放大器→差分放大器→集成运算放大器→频率响应与稳定性→负反馈→功率放大器→直流电源。内容与复旦《模拟电子线路》（MICR130002）高度重叠，工程应用案例丰富，适合希望兼顾理论严谨性与实际应用能力的学习者。
