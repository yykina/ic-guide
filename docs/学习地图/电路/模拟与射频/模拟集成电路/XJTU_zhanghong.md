# 西安交通大学：模拟CMOS集成电路设计（张鸿）

## 课程简介

- 所属大学：西安交通大学
- 先修要求：模拟电子线路（差分对、运放基础）、半导体器件物理
- 编程语言：无（仿真实验可选 SPICE）
- 课程难度：🌟🌟🌟🌟🌟
- 预计学时：80 小时

张鸿教授在西安交通大学长期讲授《模拟CMOS集成电路设计》，其 B 站课程以 Razavi《Design of Analog CMOS Integrated Circuits》（第 2 版）为主线教材，是国内最系统的模拟集成电路中文课程之一。课程在中文语境下对 Razavi 教材逐章精讲，并补充大量国内学生常见疑难点的解析，是 Razavi 原版视频（英文）的有力中文辅助资料。

课程内容覆盖：MOSFET 精确模型与二阶效应→单级放大器（CS/CG/CD 及其变体）→电流镜与参考源（Cascode、Wilson、Widlar）→差分对与差分放大器→运算放大器设计（单级/折叠 Cascode/两级 OTA）→噪声分析（热噪声、闪烁噪声、噪声系数）→频率响应与稳定性（极点/零点、相位裕度）→反馈与补偿（密勒补偿、前馈补偿）→带隙基准电路。

对于以中文为母语、希望在进入复旦《模拟集成电路设计原理》（MICR130030）前夯实基础的学生，本课程是衔接模拟电子线路与模拟IC设计的中文过渡材料。
