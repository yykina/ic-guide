# 东南大学 吴金：CMOS 模拟集成电路设计

## 课程简介

- 所属大学：东南大学
- 先修要求：模拟电子线路、半导体器件基础
- 编程语言：无（仿真可配合 SPICE / Cadence）
- 授课教师：吴金
- 课程难度：🌟🌟🌟🌟
- 预计学时：70 小时（全 48 讲）

吴金老师是东南大学集成电路方向的资深教师，这套《CMOS 模拟集成电路设计》全程录像是 B 站流传最广的中文模拟 IC 课之一，以“讲透直觉”著称。48 讲从“什么是模拟 IC、怎么学模拟 IC”开场，依次讲器件（PN 结、BJT、MOSFET 模型与二阶效应）、偏置与电流镜、Cascode 结构、基准源及其改进、共源/共漏放大器及其频域与阶跃响应、多级与 push-pull 输出级、差分放大器与各类改进、运放设计、频率响应、稳定性与相位裕度、密勒补偿（补偿电容与调零电阻）、闭环极点与频率补偿设计方法，最后覆盖比较器、迟滞比较器与振荡器。

与 Razavi 教材逐章精讲的西交张鸿课相比，吴金老师更偏工程师视角，大量时间花在“这个结构为什么长这样、改进它的思路从哪来”，对建立模拟设计的物理直觉非常有效。两门课互补，建议以一门为主线、另一门查漏。学完后再读 Razavi《Analog CMOS》会顺很多，也能直接衔接复旦《模拟集成电路设计原理》（MICR130030）。
