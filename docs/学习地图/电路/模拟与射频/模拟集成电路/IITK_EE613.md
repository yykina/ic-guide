# EE613: High-Frequency Analog Circuit Design

## 课程简介

- 所属大学：IIT Kanpur（YouTube 公开系列，SSCD IIT Kanpur 频道）
- 先修要求：模拟电子线路（BJT/MOSFET 放大器）、基础电路分析、信号与系统
- 编程语言：无
- 课程难度：🌟🌟🌟🌟
- 预计学时：80 小时

EE613 是 IIT Kanpur 开设、在 YouTube 公开的高频模拟电路课程，是该方向内容最完整的公开课系列之一，每年更新新录制版本（2024、2025 年均有），长期维护。课程系统讲授高频段模拟集成电路设计的核心议题：单级与多级运算放大器在高频段的特性分析（米勒效应、主极点/非主极点、GBW）、噪声建模（热噪声、闪烁噪声）与最小化设计、元器件失配的统计分析及对电路精度的影响，以及各类高频非理想效应的量化与处理方法。

EE613 侧重于模拟集成电路在“高频但非射频”区间（数十 MHz 到数 GHz 以下）的设计挑战，是从模拟电子线路/模拟集成电路设计过渡到射频集成电路（Razavi RF 课程）之前的重要桥梁。建议在复旦 MICR130030（模拟集成电路设计原理）之后选修。
