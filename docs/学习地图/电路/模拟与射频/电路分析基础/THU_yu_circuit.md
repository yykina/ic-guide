# 清华大学 于歆杰：电路原理（国家精品）

## 课程简介

- 所属大学：清华大学
- 先修要求：微积分、线性代数、大学物理（电磁学）
- 编程语言：无
- 授课教师：于歆杰
- 课程难度：🌟🌟🌟
- 预计学时：约 48-64 学时

清华首门 MOOC，国家级精品课程。于歆杰老师主讲，最大亮点是 Week 0 补充微积分/线代/物理前置知识，零基础友好。系统覆盖电阻电路、动态电路、正弦稳态、三相电路、二端口等四大模块。

是中文世界最权威的电路入门课，对标 UCB EE16A&B 的中文替代。
