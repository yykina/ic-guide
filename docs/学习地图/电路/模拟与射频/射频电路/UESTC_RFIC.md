# 电子科技大学：射频集成电路（成电慕课）

## 课程简介

- 所属大学：电子科技大学（UESTC）
- 先修要求：模拟集成电路
- 编程语言：无（含 Cadence 实验）
- 授课教师：游飞团队
- 课程难度：🌟🌟🌟🌟🌟
- 预计学时：9 个 Lecture

游飞团队建设的成电 RFIC 慕课，结构完全对标 Razavi RFIC——CMOS 工艺/器件 → 单级放大 → 噪声 → 收发机 → LNA → Mixer → 无源器件 → PA。国内 RF/MMW 团队的代表性课程，与复旦殷韵版互为补充。
