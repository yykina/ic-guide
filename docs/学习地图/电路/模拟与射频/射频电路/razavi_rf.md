# Razavi RF Microelectronics (UCLA EE164)

## 课程简介

- 所属大学：UCLA（加州大学洛杉矶分校）
- 先修要求：模拟电子线路、模拟集成电路设计原理（运放、MOSFET 基础）
- 编程语言：无（仿真实验可选 SPICE）
- 课程难度：🌟🌟🌟🌟🌟
- 预计学时：120 小时

Behzad Razavi 是模拟与射频集成电路设计领域最具影响力的学者之一，其教材《RF Microelectronics》（第 2 版，Prentice Hall）是全球 RF IC 方向的标准研究生教材。这套 YouTube 视频是 Razavi 在 UCLA 讲授 EE164（RF Microelectronics）的完整课程录像，与教材章节一一对应，是目前公开可获取的最完整的射频集成电路学习地图。

课程内容完整覆盖射频 IC 设计的全链路：收发机系统架构（超外差、零中频）→ 噪声与非线性分析（噪声系数 NF、IIP3、IIP2）→ 低噪声放大器（LNA）设计 → 混频器（无源/有源）→ 振荡器与 VCO（相位噪声建模）→ 功率放大器（PA，A/B/C 类）→ 锁相环（PLL）与频率合成器。
