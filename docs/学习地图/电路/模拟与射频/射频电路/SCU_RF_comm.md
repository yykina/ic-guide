# 四川大学 刘长军：射频通信电路（精品共享课）

## 课程简介

- 所属大学：四川大学
- 先修要求：高频电子线路或同级
- 编程语言：无（含 CAD 设计实验）
- 授课教师：刘长军
- 课程难度：🌟🌟🌟🌟
- 预计学时：一学期

介于”高频电子线路”和”RFIC”之间的衔接桥梁——从传输线集总模型、CAD 设计平台到 RF 放大器/功放设计，把分立元件高频电路推到工程实战。

适合在西电高频之后、进入射频 IC 之前作为过渡。
