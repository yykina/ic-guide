# 西安电子科技大学：高频电子线路（国家精品）

## 课程简介

- 所属大学：西安电子科技大学
- 先修要求：电路分析、模拟电子线路
- 编程语言：无
- 授课教师：曾兴雯、陈健团队；MOOC 由邓军、赵建勋主讲
- 课程难度：🌟🌟🌟🌟
- 预计学时：约 64 学时

国内高频电子线路国家级精品课，曾兴雯/陈健团队建设，配套高教版曾兴雯教材。MOOC 由邓军、赵建勋主讲（知乎口碑“如沐春风”）。覆盖小信号选频放大、调幅/调频、混频、振荡器、PLL 全套高频电路。

是 EE613 / MIT 6.776 的中文对应入门课。
