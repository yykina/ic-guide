# 〔待建〕功率电子（DC-DC/栅驱动/PMIC）

> 这个槽位虚位以待。如果你熟悉该领域，欢迎通过[参与建设](../../../../参与建设.md)推荐课程，一起完善这份学习地图。

## 为什么需要它

服务方向：[功率半导体与宽禁带器件](../../../../科研方向/功率半导体与宽禁带器件.md)。

把 SiC/GaN 器件用起来的电路学问：开关变换器拓扑、栅驱动、软开关、电源管理 IC。功率器件方向的同学缺了这块就只懂器件不懂应用。
