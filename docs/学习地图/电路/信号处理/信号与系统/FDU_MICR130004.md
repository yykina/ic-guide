# 复旦：信号与系统

## 课程简介

- 所属大学：复旦大学
- 课程代码：ICSE20006（2025 培养方案）/ MICR130004（2021 方案）
- 先修要求：数学分析、线性代数
- 编程语言：无（部分实验可用 MATLAB）
- 课程难度：🌟🌟🌟🌟
- 预计学时：64 学时

信号与系统是电子信息类专业的核心基础课，贯通连续时间域（傅里叶/拉普拉斯变换）与离散时间域（Z 变换/DTFT），是后续通信、数字信号处理、自动控制的共同前置。复旦微电子学院开设此课，与电子系同名课程内容相近，面向 IC 方向学生强调频域思维与系统稳定性的建立。
