# 复旦：数据结构与算法

## 课程简介

- 所属大学：复旦大学
- 课程代码：ICSE30011（2025 培养方案）/ MICR130009（2021 方案）
- 先修要求：程序设计基础（C/C++ 或 Python）
- 编程语言：C/C++
- 课程难度：🌟🌟🌟
- 预计学时：48 学时

数据结构与算法是计算机与电子工程方向的必修基础课。复旦微电子学院开设此课，旨在培养 IC 工程师在 EDA 工具开发、算法优化及嵌入式软件编写中所需的编程能力。核心内容包括线性表、树、图、排序与查找算法及复杂度分析。
