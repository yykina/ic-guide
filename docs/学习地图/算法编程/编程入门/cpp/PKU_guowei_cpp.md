# 北京大学 郭炜：程序设计与算法（三）C++ 面向对象

## 课程简介

- 所属大学：北京大学
- 先修要求：C 语言
- 编程语言：C++
- 授课教师：郭炜
- 课程难度：🌟🌟🌟
- 预计学时：8 周

郭炜老师在北大讲授十五年的 C++ 课。期末大作业是《魔兽世界》游戏模拟程序（非常硬核），配套教材《新标准 C++ 程序设计》。

是中文圈最被推崇的 C++ 系统课之一，适合先学完 C 后进阶 C++。
