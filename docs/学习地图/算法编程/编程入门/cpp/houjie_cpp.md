# 侯捷：C++ 面向对象高级开发

## 课程简介

- 所属大学：博览网 / 侯捷
- 先修要求：C++ 入门
- 编程语言：C++
- 授课教师：侯捷
- 课程难度：🌟🌟🌟🌟
- 预计学时：上 13 讲 + 下若干讲

台湾 C++ 大师侯捷的经典课程。从 `Complex` / `String` 两个 class 范例切入构造/拷贝/虚函数/继承，中文圈最被推崇的“读懂 C++ 类”的课。

适合学完基础 C++ 后想真正吃透面向对象的同学。
