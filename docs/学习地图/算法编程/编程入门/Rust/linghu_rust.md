# 程序员令狐壹冲：Rust 编程视频教程

## 课程简介

- 所属大学：B 站 UP “程序员令狐壹冲”
- 先修要求：任一编程语言基础
- 编程语言：Rust
- 授课教师：令狐壹冲
- 课程难度：🌟🌟🌟
- 预计学时：基础 + 进阶 + 项目实战三部

国内 Rust 中文社区高人气讲师，覆盖基础语法到网络编程、WebAssembly 与区块链项目实战。是杨旭课之外最常被推荐的系统性中文 Rust 教程。
