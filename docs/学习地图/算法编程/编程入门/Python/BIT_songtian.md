# 北京理工大学 嵩天：Python 语言程序设计（国家精品）

## 课程简介

- 所属大学：北京理工大学
- 先修要求：无
- 编程语言：Python
- 授课教师：嵩天
- 课程难度：🌟🌟
- 预计学时：11 周（约 30 学时）

国家精品课程，累计学习者超 550 万。嵩天老师以“理解和运用计算生态”为理念的零基础 Python 入门标杆，配套自研 Python123 实践平台。

适合零编程基础同学作为 CS50P / UCB CS61A 之前的中文起步课。
