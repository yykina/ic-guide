# 北京大学 陈斌：数据结构与算法 Python 版

## 课程简介

- 所属大学：北京大学
- 先修要求：Python 入门
- 编程语言：Python
- 授课教师：陈斌
- 课程难度：🌟🌟
- 预计学时：16 周课堂实录

陈斌老师在北大长期讲授的公共基础课课堂实录，从计算思维到 Python 语法再到数据结构一条龙。是国内最完整的“以 Python 为载体的计算导论”课程，适合非 CS 专业但想系统学编程的同学。
