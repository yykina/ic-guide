# 上海交大:传热学

## 课程简介

- 所属大学：上海交通大学
- 先修要求：高等数学、大学物理
- 编程语言：无
- 任课教师：赵长颖、王平阳、欧阳华、洪芳军
- 课程难度：🌟🌟🌟🌟
- 预计学时：52 学时

上海交大动力工程及工程热物理系的传热学课程,**国家级精品课程之一**。教学团队由 4 位教授组成,分别主讲导热、对流、辐射、综合应用四个板块,师资力量强大。配套教材使用[西安交大陶文铨](XJTU_taowq.md)团队主编的《传热学》(第四版),与该课形成互补。

课程内容覆盖:**基本概念 + 热传导 + 热对流 + 热辐射 + 综合应用**,五大模块的标准教学序列。相比西交陶院士的版本,上交版的工程案例选取更偏向制造业(船舶、动力机械、能源),**对做封装/散热实战的同学也有参考价值**。
