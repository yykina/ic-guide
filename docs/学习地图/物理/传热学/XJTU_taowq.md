# 西安交大 陶文铨:传热学

## 课程简介

- 所属大学：西安交通大学
- 先修要求：高等数学、大学物理
- 编程语言：无(进阶可用 MATLAB / COMSOL)
- 任课教师：陶文铨(中国科学院院士)、王秋旺
- 课程难度：🌟🌟🌟🌟
- 预计学时：65 学时(完整版)

陶文铨院士是**国内传热学领域的权威**,他在西安交大开设的《传热学》课是国家级精品课,**B 站和中国大学 MOOC 双平台公开**,在国内工科自学群体中口碑极佳。同名教材(杨世铭、陶文铨主编《传热学》)是国内传热学教学的事实标准——上海交大、哈工大等校的传热学课程也用同一本教材。

课程系统覆盖**导热、对流、辐射三大块,以及综合工程应用与换热器设计**。陶院士讲法严谨而不晦涩,推导清晰,案例选取多来自电力/能源/航空工业实际,**做芯片散热的同学完全能从中找到对应的工程方法论**。
