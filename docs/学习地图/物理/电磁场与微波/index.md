## 为什么需要它

服务方向：[射频与毫米波IC](../../../科研方向/射频与毫米波IC.md)、[光电子与硅光集成](../../../科研方向/光电子与硅光集成.md)。

从静电场、麦克斯韦方程到传输线、S 参数、微波网络，是射频电路和光波导的共同物理底座。没有它，Razavi RF 里的阻抗匹配和硅光里的波导模式都只能死记硬背。
