# 北大 周治平:光电子学

## 课程简介

- 所属大学：北京大学
- 先修要求：大学物理(电磁学、波动光学)
- 编程语言：无
- 任课教师：周治平(北大信息科学技术学院教授,《硅基光电子学》一书作者)
- 课程难度：🌟🌟🌟🌟
- 预计学时：32 讲(完整版)

周治平教授是**中国硅基光电子学的奠基人之一**,北大信息学院教授,《硅基光电子学》(科学出版社)一书作者。他主张用“硅基光电子”而非“硅光”,理由:**硅光只是在硅上做单个光电器件,不能完全反映硅基光电集成技术的覆盖范围**——后者要做异质集成、光电协同设计,这是研究方向的关键认识。

这门《光电子学》是周教授在北大讲了多年的核心课,B 站有完整 32 讲录像公开。课程系统梳理光电子学的物理基础和器件原理,**比起单纯的“光学课”,更面向 IC 学生关心的工程器件层面**:

- 光的电磁波本质 + 在媒质中的传播
- 半导体材料的光学性质
- 激光器原理(LD)、发光二极管(LED)
- 调制器(EAM、EOM)、探测器(PD、APD)
- 光波导基础(slab/strip/rib)
- 集成光路与硅基集成
