# MIT 8.03: Vibrations and Waves

## 课程简介

- 所属大学：MIT
- 先修要求：MIT 8.01/8.02(力学 + 电磁学)、微积分基础
- 编程语言：无
- 任课教师：Walter Lewin / Yen-Jie Lee
- 课程难度：🌟🌟🌟
- 预计学时：约 36 学时

MIT 8.03 是 MIT 物理三部曲的最后一部(8.01 力学 → 8.02 电磁 → 8.03 波动),专攻**所有形式的波**——简谐振动、机械波、电磁波、光波。Walter Lewin 教授的演示式教学风格让这门课成为 YouTube 上最受欢迎的物理课之一:用绳子做横波演示、用弹簧讲简谐运动、用激光器演示干涉衍射图样,**直观到令人印象深刻**。

对 IC/光电子方向的同学来说,8.03 是衔接“大物中的光学”到“光电子学/集成光子学”的关键一步——它把光当成电磁波系统讲清楚:相速度、群速度、色散关系、波导模式分析的概念底子都在这里。学过这门后,再看硅光波导的色散曲线就不会一头雾水。
