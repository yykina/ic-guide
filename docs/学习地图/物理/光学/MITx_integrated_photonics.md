# MITx: Fundamentals of Integrated Photonics

## 课程简介

- 所属大学：MIT(via edX / AIM Photonics Academy)
- 先修要求：电磁波基础(大学物理或 MIT 8.03 即可)
- 编程语言：无
- 任课教师：Lionel Kimerling 团队 / AIM Photonics Academy 讲师
- 课程难度：🌟🌟🌟
- 预计学时：自定步调,核心内容约 30-40 学时

这是 MIT 在 edX 上发布的**硅光子入门课**,由 MIT 电子材料中心 (EMC²) 与美国国家光子制造研究院 AIM Photonics 联合制作。课程目标非常明确:让学生在不深入物理细节的前提下,**快速理解硅光集成的核心概念、设计流程和应用场景**。对一个想做硅光研究但不愿先啃半年波动光学的 IC 学生来说,这是**入门最高效的路径**。

模块化设计:从“光学材料导论”到“集成光子器件概览”,再到“光链路设计与系统驱动”。课程后半部分介绍如何用 PDK(Process Design Kit)在代工厂(如 AIM Photonics)上设计一个 Mach-Zehnder 调制器(MZM)——**这就是硅光研究的“hello world”**。

## 涵盖内容

| 模块 | 主要内容 |
|---|---|
| 光学材料导论 | 半导体、电介质、聚合物的光学性质 |
| 光的传播 | 几何光学 → 电磁光学 → 导波光学 |
| 光子器件 | 调制器、探测器、激光器、波导耦合器 |
| 集成与封装 | 芯片级集成 + 光纤接口 + 热管理 |
| 应用驱动 | telecom / datacom / 光互连 / 光计算 |

## 备注

- 课程设计目标是让本科生 / 行业工程师能跟上,数学要求不高,不必因 “PhD course” 标签却步
- 课程后续的 **Photonic Integrated Circuits 1** 才是更深入的器件物理,可选
- 想接触实际设计:配合 UBC 的 [SiEPIC 开源工具](https://github.com/SiEPIC) 走一遍 MZM 设计流程,**0 成本就能实战**
