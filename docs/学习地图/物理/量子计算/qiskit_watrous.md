# Qiskit：Understanding Quantum Information and Computation（Watrous）

## 课程简介

- 主讲：John Watrous（University of Waterloo，量子信息理论权威）
- 出品：IBM Quantum / Qiskit
- 先修要求：线性代数、概率论
- 课程难度：🌟🌟🌟🌟
- 预计学时：15 讲

理论与实践结合的英文量子计算课，Qiskit 官方出品。Watrous 是量子信息复杂性理论领域的顶级学者，课程风格严谨但不失直觉：从量子态和测量的数学框架讲起，覆盖量子纠缠、量子电路模型、主要量子算法（Deutsch-Jozsa / Grover / QFT / Shor），最后接入 Qiskit 编程实践。配套教材以 arXiv 形式公开（IBM Quantum Learning Notes）。

与北大李彤阳课的关系：Watrous 课理论 + Qiskit 实操并重，李彤阳课侧重算法推导和复杂性；两门选一入门即可，有余力可互补。
