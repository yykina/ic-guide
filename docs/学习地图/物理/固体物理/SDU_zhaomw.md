# 山东大学 赵明文：固体物理

## 课程简介

- 所属大学：山东大学
- 先修要求：量子力学、热力学与统计物理基础、线性代数
- 编程语言：无
- 授课教师：赵明文
- 课程难度：🌟🌟🌟🌟
- 预计学时：90 小时（全 57 讲）

赵明文老师的山东大学《固体物理》全程课堂录像是 B 站上口碑最好的中文固体物理课之一，以板书推导见长，公式来龙去脉交代得非常清楚，适合自学者逐讲跟下来。课程体系是国内经典的黄昆/Kittel 路线，57 讲完整覆盖固体物理主干：晶体结构与对称性、倒格子与 X 射线衍射、晶体结合、晶格振动与声子（含长波近似与黄昆方程）、晶格热容与热输运、晶体缺陷，再到能带理论（Bloch 定理、平面波法、紧束缚法、赝势）、有效质量与半导体能带、能态密度与费米面、朗道能级与 de Haas–van Alphen 效应，最后以玻尔兹曼方程和电导率理论收尾。

对微电子学生来说，固体物理是从量子力学走向半导体物理与器件的必经一站。这门课里的能带理论、有效质量、声子与电声散射、玻尔兹曼输运，正是后续半导体物理、器件物理和芯片热管理课程反复调用的底层语言。如果你打算认真补一遍固体物理，这门中文全程课是目前可得的最稳妥选择。
