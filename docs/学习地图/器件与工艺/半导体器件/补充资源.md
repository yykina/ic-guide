# 半导体器件：补充资源

## 英文配套课程

### NPTEL：IIT Roorkee 微电子——从器件到电路

- 平台：NPTEL（免费）
- 课程主页：<https://nptel.ac.in/courses/108107142>（Microelectronics: Devices to Circuits，Sudeb Dasgupta 教授，约 60 讲）
- 课程视频（YouTube 官方播放列表）：<https://www.youtube.com/playlist?list=PLLy_2iUCG87C-GS7B4vJ37Iz3fevmYU1h>
- 简介：从 MOSFET/BJT 器件物理讲到模拟与数字电路分析，“器件 + 电路”一体化结构，配合 Sedra/Smith 或 Neamen 教材使用效果良好

### 中国大学 MOOC：国内高校半导体器件课程

- 西安电子科技大学《半导体器件物理 I》：<https://www.icourse163.org/course/XIDIAN-1206896809>
- 西安电子科技大学《半导体器件物理 II》：<https://www.icourse163.org/course/XDU-1472033161>
- 东南大学《电子器件（双语）》：<https://www.icourse163.org/course/SEU-1003359011>
- 简介：与国内主流教材（刘恩科《半导体物理学》体系）配套，适合作为中文入门参考
