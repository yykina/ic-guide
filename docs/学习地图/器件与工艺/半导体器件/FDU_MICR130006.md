# 复旦：半导体器件原理

## 课程简介

- 所属大学：复旦大学
- 课程代码：ICSE30020（2025 培养方案）/ MICR130006（2021 方案）
- 先修要求：半导体物理、量子力学基础
- 课程难度：🌟🌟🌟🌟🌟
- 授课教师：蒋玉龙、茹国平等
- 预计学时：64 学时

半导体器件原理是复旦微电子学院器件方向的核心专业课，从半导体物理出发推导 pn 结、双极型晶体管（BJT）、MOS 电容与 MOSFET 的工作原理与电流-电压特性，并延伸至短沟道效应、速度饱和等现代 CMOS 器件议题，是模拟电路、数字集成电路设计的直接物理基础。
