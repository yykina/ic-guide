# 哈尔滨工业大学 王蔚：微电子工艺

## 课程简介

- 所属大学：哈尔滨工业大学
- 先修要求：半导体物理、半导体器件基础
- 编程语言：无
- 授课教师：王蔚
- 课程难度：🌟🌟🌟
- 预计学时：80 小时（全 106 讲）

哈工大王蔚教授的《微电子工艺》全程录像，是 B 站上讲数最全的中文集成电路工艺课。106 讲系统覆盖硅片制备、热氧化、光刻、刻蚀、扩散与离子注入、薄膜淀积（CVD/PVD）、金属化与互连等完整工艺模块，并讲解各工序之间如何串成一条完整的 CMOS 工艺流水线。讲法贴近产线实际，对每种工艺的设备、参数与缺陷模式都有交代。

对微电子学生来说，这门课是复旦《集成电路工艺原理》（MICR130007）最对口的中文视频补充。课内讲义跟不上或想反复听某个工艺模块时，可以按分 P 标题精确定位到对应工序。学完后再去读 IEDM 或 VLSI Symposium 的先进工艺论文，对 FinFET/GAA 流程的理解会顺很多。
