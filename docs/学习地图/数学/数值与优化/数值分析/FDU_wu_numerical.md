# 复旦大学 吴宗敏：数值逼近

## 课程简介

- 所属大学：复旦大学数学系
- 先修要求：数学分析、线性代数
- 编程语言：MATLAB（可选）
- 授课教师：吴宗敏
- 课程难度：🌟🌟🌟🌟
- 预计学时：42 讲

复旦数学系吴宗敏教授数值逼近课，国内函数逼近方向的代表性课程，是数值分析中**插值 / 逼近 / 最小二乘**部分最深入的中文资源。相比通用“数值分析”课，更聚焦于逼近论的理论基础（Bernstein 多项式、Chebyshev 节点、样条函数等）。

是 MIT 18.330 的“理论深度补充”——MIT 18.330 偏算法工程，吴宗敏课偏数学原理。
