# 中国科学技术大学：数值分析（38 讲）

## 课程简介

- 所属大学：中国科学技术大学
- 先修要求：数学分析、线性代数
- 编程语言：MATLAB / Python（实验）
- 授课教师：中科大数学系
- 课程难度：🌟🌟🌟
- 预计学时：38 讲

中科大数学系本科数值分析全程录像，系统讲解**误差分析 / 插值 / 数值积分 / 线性方程组迭代 / 非线性方程求解 / ODE 数值解**。节奏适中，证明清晰，是 MIT 18.330 的中文打底课。

原 Youku 38 讲版本，多人搬运至 B 站，可作为入门数值分析的中文首选。
