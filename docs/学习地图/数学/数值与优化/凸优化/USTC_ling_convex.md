# 中科大 凌青：最优化理论（凸优化）

## 课程简介

- 所属大学：中国科学技术大学（凌青老师，现中山大学）
- 先修要求：高等数学、线性代数、概率论
- 编程语言：MATLAB / Python（CVXPY）
- 授课教师：凌青
- 课程难度：🌟🌟🌟🌟
- 预计学时：约 60 学时

B 站中文凸优化课程的事实标杆。凌青老师讲授，内容编排与 Stephen Boyd 教材《Convex Optimization》一致，但用中文逐步推导每一个证明，是 Stanford EE364A 的最佳中文配套。

视频已 4K 修复，CSDN 有 55 篇完整学习笔记可对照。
