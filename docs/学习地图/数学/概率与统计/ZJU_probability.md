# 浙江大学：概率论与数理统计

## 课程简介

- 所属大学：浙江大学
- 先修要求：微积分
- 编程语言：无
- 授课教师：张帼奋 等（浙江大学 MOOC 团队）
- 课程难度：🌟🌟
- 预计学时：60 小时（绪论 + 64 讲，每讲拆分为短视频）

这是浙江大学在中国大学 MOOC 上的《概率论与数理统计》全程课，与国内最通行的盛骤等编《概率论与数理统计》（浙大版）教材体系完全对应。课程共九章，从概率的基本概念、古典概型讲起，覆盖一元与多元随机变量及其分布、数字特征、大数定律与中心极限定理，然后进入数理统计部分，讲抽样分布、点估计与区间估计、假设检验，最后以方差分析与一元线性回归收尾。每讲拆成十几分钟的短视频，节奏平缓，跟着浙大版教材逐章推进毫无障碍。

对微电子学生来说，这门课适合作为中文打底课。如果英文课跟起来吃力，或者需要快速对齐国内考研与课内考试的知识框架，先用它把概率与统计的主干过一遍，再去看 Stat 110 或 6.041 补直觉，是性价比很高的路线。统计部分的估计与假设检验，正是器件参数提取、工艺良率分析这些场景每天都在用的工具。
