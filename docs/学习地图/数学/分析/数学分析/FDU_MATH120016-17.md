# 复旦：数学分析B

## 课程简介

- 所属大学：复旦大学
- 课程代码：MATH120016-17（2021 方案）；2025 培养方案对应必修为[高等数学A](FDU_MATH10015-16.md)，本课是更严格的替代
- 先修要求：无
- 编程语言：无
- 任课教师：谢锡麟 / 陈纪修

同一门课，两位老师两种讲法，按自己的口味选一位跟完即可。

=== "谢锡麟"

    - 课程难度：🌟🌟🌟🌟🌟

    谢老师的数学分析在复旦堪称一绝，在国内高校纷纷采用欧美教材的当下，谢老师承袭自苏联的教学体系可谓独树一帜。谢老师对展开有着非常深入的研究，其教授的展开方法我多年以后还在使用，可谓一招鲜吃遍天。该课程难度远超其他校内同名课程，认真学习可以打下坚实的数学分析基础。

    - 视频：[力学数学-谢锡麟](https://space.bilibili.com/589977552/lists)

=== "陈纪修"

    - 课程难度：🌟🌟🌟🌟

    陈老师是复旦数学系的泰斗，其编修的《数学分析》可以说是内地最通俗易懂又不失严谨的教材了。与谢老师“从一般到特殊”的教学方式不同，陈老师遵循“从特殊到一般”的顺序，从一些特殊例子出发，一步一步引导我们推导出一般的定理，这更加符合正常人类的学习曲线，学起来也更有成就感。

    - 视频：[数学分析-陈纪修](https://www.bilibili.com/video/BV15v411g7VP/?spm_id_from=333.337.search-card.all.click)
