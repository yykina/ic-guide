# 西安交通大学：复变函数（国家精品）

## 课程简介

- 所属大学：西安交通大学
- 先修要求：高等数学（微积分）
- 编程语言：无
- 授课教师：西安交大数学团队（国家级教学团队）
- 课程难度：🌟🌟🌟
- 预计学时：32 讲（主流版）/ 68 讲（细化版）

西安交大国家级精品课程，国家级教学团队打造，配套教材是王绵森“十一五”国家级规划教材《复变函数》。B 站口碑最稳的正规中文复变课，覆盖复数 / 复变函数 / 复积分 / 留数定理 / 共形映射全套。
