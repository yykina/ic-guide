# 北京邮电大学：现代密码学（国家精品）

## 课程简介

- 所属大学：北京邮电大学
- 先修要求：抽象代数、数论基础
- 编程语言：无（理论课）
- 授课教师：北邮信息安全团队
- 课程难度：🌟🌟🌟
- 预计学时：约 48 学时

国家级精品在线开放课程，是 B 站中文圈最系统的现代密码学课程。覆盖流密码、分组密码（DES/AES）、公钥密码（RSA/ElGamal/ECC）、数字签名、Hash 函数、密码协议全栈，配套教材是杨波《现代密码学》（清华版）——国内密码学的事实教材。

与 Stanford Boneh 课形成“中文系统讲解 vs 英文权威讲解”的互补。
