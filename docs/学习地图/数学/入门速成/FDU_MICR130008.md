# 复旦：工程数学及概率方法

## 课程简介

- 所属大学：复旦大学
- 课程代码：ICSE20009（2025 培养方案）/ MICR130008（2021 方案）
- 先修要求：数学分析、线性代数
- 编程语言：无
- 任课教师：荆明娥
- 课程难度：🌟🌟🌟

复旦微电子学生在学完数学分析和线性代数后唯一一门也是最后一门数学课，讲得比较简单，也比较枯燥，但很有用。其中涉及到的复变函数和积分变换是信号处理的基础，而概率论则是打开机器学习和深度学习大门的钥匙。
