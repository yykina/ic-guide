# 北京大学：离散数学（国家精品）

## 课程简介

- 所属大学：北京大学
- 先修要求：高中数学
- 编程语言：无
- 授课教师：屈婉玲、耿素云、王捍贫、刘田
- 课程难度：🌟🌟🌟
- 预计学时：134 集 / 约 120 学时

屈婉玲老师主持的北大国家精品课程，B 站完整覆盖**六大板块**：数理逻辑、集合论、代数结构、组合数学、图论、初等数论。配套教材是北大经典版黑皮书《离散数学》。

国内离散数学最权威的中文公开课，与 UCB CS70 / CS126 互为对照——北大版偏向数学严谨性，CS70 偏向 CS 应用，结合学效果最佳。
