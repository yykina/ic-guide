# 南京大学 孙智伟：近世代数（国家级一流课程）

## 课程简介

- 所属大学：南京大学
- 先修要求：高等代数（线性代数 + 多项式理论）
- 编程语言：无
- 授课教师：孙智伟
- 课程难度：🌟🌟🌟🌟
- 预计学时：104 讲 / 约 60-80 学时

孙智伟教授 20 余年抽象代数教学积累的国家级一流本科课程（2025），系统覆盖群-环-域三大代数结构。配套教材是孙智伟编《近世代数》（南大社），从置换群、对称性、Sylow 定理一路推到 Galois 理论。

讲法偏国内数学系传统风格，定理-证明-例题节奏稳，是中文圈最系统的抽象代数公开课，可作为 MIT 18.701/702 (Artin) 的中文对照参考。
