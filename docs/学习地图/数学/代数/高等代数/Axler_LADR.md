# Axler：Linear Algebra Done Right（官方配套视频）

## 课程简介

- 所属大学：San Francisco State University（教材作者 Sheldon Axler 本人录制）
- 先修要求：一学期线性代数（会算矩阵即可），有一定证明阅读能力
- 编程语言：无
- 授课教师：Sheldon Axler
- 课程难度：🌟🌟🌟🌟
- 预计学时：50 小时

《Linear Algebra Done Right》是英语世界最有影响力的第二门线性代数教材，从抽象向量空间和线性映射出发重建整个理论，刻意把行列式压到最后，主线直奔特征值、内积空间、谱定理与 Jordan 形。第四版已开放获取，官网可免费下载全书 PDF。Axler 本人为教材录制了完整配套视频，每章一组短视频并附 slides，第三版与第四版均有对应版本，YouTube 官方播放列表与官网同步。

对微电子学生来说，这是中文丘维声课之外的英文选项。它的价值不在算矩阵，而在把“线性算子的结构”讲成一套清晰的语言。谱定理与正交分解是信号处理、量子力学和数值方法的共同地基。视频是教材的伴读而非独立授课，最佳用法是读一节书、看对应一段视频、做书后习题。
